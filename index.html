About Me

The Story Behind the Screen

Hey! I am Poor YouTuber. Despite the name, I build premium digital content. I started this channel to document my self-taught coding path and break down complex engineering architectures into easy-to-digest visual guides.

My goal is to make programming education less intimidating and show that you do not need expensive hardware or degrees to create world-class software.

  • ✓

    Practical Tutorials First

    No unnecessary theoretical padding—just raw projects, clean files, and standard coding patterns.

  • ✓

    Open-Source Commitment

    All source code from my video series is shared freely on GitHub for the community to study.

Year 1: Launch

Started with screen capture software and a budget headset. Focused on core HTML/CSS concepts.

Year 2: 10K Subs

Built a dedicated community and expanded content to Javascript frameworks and web animations.

Year 3: Full-Time

Left my traditional gig to build tutorials, courses, and open source templates full time.

The Horizon

Aiming to launch interactive learning platforms and help millions code their dreams.
